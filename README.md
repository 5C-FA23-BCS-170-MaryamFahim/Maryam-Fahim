TASK Mania Project Title: Task Mania Platform: Flutter Database: SQLite Submitted By: Maryam Fahim (FA23-BCS-170)  Course: Mobile Application Development Instructor: Sir Abrar Siddique

Description: Task Mania is a mobile-based task management system designed to help users manage their daily activities efficiently. The app allows adding, updating, and deleting tasks, marking them as completed, and storing data locally through SQLite for offline use.

Key Features:

CRUD operations (Create, Read, Update, Delete) Local storage using SQLite Interactive and simple user interface Categorize tasks by priority or date Store data locally using SQLite Simple and user-friendly interface

Tools Used: Flutter (Dart), SQLite, Android Studio

Flutter (Dart) for frontend and app logic SQLite for offline data storage Material Design for UI components

Installation:

Download the latest APK from the releases section or from "base.apk".
Install the APK on your Android device.
Launch Task Mania and start managing your tasks instantly!
